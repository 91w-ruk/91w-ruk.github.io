# 🤝 [陪伴向：《病房里的向日葵》](https://91w-ruk.github.io/)

陈默推开病房门的时候，看见林曦正对着窗外的阳光笑，她的头发因为化疗掉得差不多了，脸上却带着淡淡的红晕。

三年前，林曦被确诊为乳腺癌，陈默在她最绝望的时候，向她求婚。“林曦，”他握着她的手，“不管你变成什么样，我都想和你在一起。” 化疗的日子很苦，林曦掉光了头发，吃什么吐什么，有时候会对着镜子哭。陈默就每天给她讲笑话，给她读她最喜欢的书，在她睡着的时候，轻轻抚摸她的额头。 他在病房的窗台上摆了一盆向日葵，每天早上都会换新鲜的水。“向日葵跟着太阳转，”他说，“你也要跟着希望走，我就是你的太阳。”

林曦手术那天，陈默在手术室外守了八个小时，手里攥着一朵向日葵。当医生出来说手术成功时，他蹲在地上，哭得像个孩子。 后来林曦慢慢康复，他们在一个晴朗的日子里补办了婚礼。婚礼上，林曦穿着白色的婚纱，手里捧着向日葵，陈默看着她，眼里满是温柔。 现在他们已经一起走过了五年，陈默每天还是会给她买向日葵，林曦也会每天给陈默做他喜欢吃的红烧肉。他们说，爱情不是轰轰烈烈的誓言，是在你最难的时候，有人陪你一起扛；是在你康复后，有人陪你一起看日出日落。原来最好的爱情，是陪伴，是不离不弃，是我和你，一起把苦难，过成幸福。
